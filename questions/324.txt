Оператор цикла общего вида, то есть оператор, вводимый служебным словом <b>for</b>, имеет следующий формат: <b>for (I; С; Е) S</b>, где 
<b>I</b> — инициализатор цикла вида «<code>int a = 0</code>»; 
<b>С</b> — выражение-условие; 
<b>Е</b> — выражение без операции запятая; 
<b>S</b> — оператор (тело цикла). 
Правильная последовательность вычисления и обработки выражений <b>I; С; Е</b> и выполнения оператора <b>S</b> может быть представлена с помощью: 
1) do { I; S; Е; } while (С); 
2) I do { S; Е; } while (С); 
3) I while (С) { S; Е; }
4) I do { if (!C) break; S; E; } while(true);
5) I do { if (!С) соntinue; S; E; } while(true);
